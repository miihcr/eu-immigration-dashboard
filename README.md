# EU Immigration Dashboard

This repository contains an R/Shiny project that explores immigration
to European countries using Eurostat data (`migr_imm8`, `demo_pjan`)
and geospatial NUTS0 boundaries. It includes an interactive Shiny
application with time-series by country and choropleth maps of both
immigration counts and immigration as a share of the population.

Developed as part of the EMOS core module during my MSc Statistics & Data Science.
